ORCycle is an Android Application

This Android Application helps you track your bike travel and keeps a record for future view.

How can ORCycle benefit you?

Ride's Data

ORCycle helps you keep a record of your bike ride data so that you can view it later to see where you have travelled.

Map view (UI)

ORCycle gathers your location via GPS and later populates the coordilates into the Google Map.

Help Build better bike route

During the trip if you find pot holes, or good tracks then send us a feedback so that we can improve it or add it in our featured list.

Follow project

If you believe we are doing amazing work then please follow our project on Favebook, Twitter or Google+.
